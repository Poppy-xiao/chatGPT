安装环境 

pip3 install revChatGPT 

通过 https://chat.openai.com/chat 注册并登录。
打开浏览器开发者工具，切换到 Application 标签页。
在左侧的 Storage - Cookies 中找到 __Secure-next-auth.session-token 一行并复制其值。
填写在代码里，重启星乃就好了

发送 gpt +内容就能获得chatGPT的回复了。
